#  Eletricitys

Sistema de monitoramento do consumo de energia elétrica desenvolvido como Projeto Integrador do SENAI.

##  Objetivo

Auxiliar usuários no acompanhamento do consumo de energia elétrica em tempo real, permitindo maior controle dos gastos e incentivo ao uso consciente da energia.

##  Tecnologias

- Python
- PyQt5
- Qt Designer
- SQLite
- ESP32
- QtCharts

##  Funcionalidades

- Cadastro de usuários
- Login
- Dashboard com gráficos
- Monitoramento do consumo
- Cálculo do custo da energia
- Histórico de consumo
